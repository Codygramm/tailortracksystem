# Image Guide for TailorTrack Website

This guide explains where to place images for your custom tailor shop website.

## Directory Structure

```
images/
├── hero/           (Hero section slideshow images)
├── about/          (About section images)
├── gallery/        (Gallery showcase images)
└── services/       (Service card images)
```

## Required Images

### Hero Section Slideshow (images/hero/)
Add 3 high-quality banner images (1920x1080px recommended):
- `tailor-shop-1.jpg` - Main shop exterior/interior
- `tailor-shop-2.jpg` - Tailor working on garments
- `tailor-shop-3.jpg` - Finished products display

### About Section (images/about/)
- `tailor-working.jpg` - Professional photo of tailor at work (800x600px)

### Gallery Section (images/gallery/)
Add 5 showcase images (600x400px recommended):
- `baju-melayu.jpg` - Finished Baju Melayu
- `baju-kurung.jpg` - Finished Baju Kurung
- `kebaya.jpg` - Finished Kebaya
- `tailor-process.jpg` - Tailoring process/workspace
- `fabrics.jpg` - Premium fabric collection

### Services Section (images/services/)
Add 6 service images (400x300px recommended):
- `baju-melayu.jpg` - Baju Melayu service
- `baju-kurung.jpg` - Baju Kurung service
- `kebaya.jpg` - Kebaya service
- `kurta.jpg` - Kurta service
- `repair.jpg` - Repair service
- `measurements.jpg` - Measurement service

## Image Guidelines

1. **Format**: Use JPG for photos
2. **Quality**: Use high-resolution images (at least 1200px width for hero)
3. **File Size**: Optimize images to be under 500KB each
4. **Style**: Professional photos with good lighting
5. **Content**: Show actual work, shop interior, fabrics, and tailoring process

## Using Placeholder Images

If you don't have images yet, you can use free stock photos from:
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)

Search terms: "tailor shop", "sewing", "fabric", "traditional clothing", "baju melayu"

## Online Image Optimization

Before uploading, optimize images at:
- TinyPNG (tinypng.com) - Compress JPG/PNG
- Squoosh (squoosh.app) - Advanced compression

## Quick Setup with Placeholder Service

You can also use placeholder images temporarily:
Replace image paths in index.php with:
```
https://via.placeholder.com/1920x1080/25344F/FFFFFF?text=Tailor+Shop
```

Just change the dimensions and text as needed.
